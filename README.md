## Intro 

An example project using the wxWidgets library and OpenGL. 

## Dependencies 

- [wxWidgets]()
- [GLAD]()

## Build Instructions 

```bash
git clone --recursive https://github.com/FrodoAlaska/WxWidgetsExample.git
cd WxWidgetsExample
```

```bash
mkdir build-debug && cd build-debug
cmake .. 
```

```bash
# Windows 
cmake --build . --config Debug

# Linux 
make 
```
